The system of automated distribution of the flow of applications in the banking sector

An automated system for distributing loan applications using optimization algorithms and machine learning.

Main features
Automatic distribution:
- Real-time processing of incoming requests
- Accounting for the qualifications and workload of employees
- Optimization of execution time

Self-learning system:
- Analysis of historical processing data
- Dynamic recalculation of the average execution time
- Adaptation to changes in the work of employees

Classification of applications:
- Correctly completed applications are distributed among the underwriters
- Incomplete or erroneous applications are sent to a specialist for clarification
- Empty or non-bank applications are automatically cancelled
